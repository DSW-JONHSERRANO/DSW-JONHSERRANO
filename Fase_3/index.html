Introducción


En la siguiente página se busca comprender el uso de las etiquetas básicas que confirman el HTML el cual, es un lenguaje de programación (enfocado en la marcación) para el contenido de las páginas web, está compuesto a base de etiquetas con el fin de ayudar a organizar la estructura del texto dentro de la página.

Temática


La temática que se empleara es por medio de ejemplos los cuales ayudaran a comprender un poco mejor los conceptos básicos de las etiquetas del HTML, atributos y valores.
